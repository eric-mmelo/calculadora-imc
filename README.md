🧮 IMC Calculator

📌 About the Project

This is a web application that calculates the Body Mass Index (BMI) based on user input (weight and height). It provides instant results along with the health classification according to BMI ranges.


🚀 Features

BMI calculation in real time

Health classification (Underweight, Normal, Overweight, Obesity)

Responsive design

Simple and intuitive interface


🛠️ Technologies Used

HTML

CSS

JavaScript


📂 How to Run

Clone the repository:

git clone https://github.com/your-username/imc-calculator.git

Open the project folder:

cd imc-calculator

Open the index.html file in your browser.
